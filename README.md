# Proyecto Mars Rover
Codigo de prueba del proyecto

El código que he subido es una prueba para el proyecto que voy a realizar de un Mars Rover. Este Mars Rover es un vehículo
de exploración espacial que fue creado por la NASA y es muy conocido a nivel mundial. En el proyecto replicaremos algunas 
funciones del Mars Rover de manera muy simplificada. Las funciones que tendrá serán las siguientes, una estación meteorológica
que constara de numerosos sensores, un brazo robótico hecho por impresión 3d y servo motores, un sistema de tracción de 4 ruedas 
y las comunicaciones como bluetooth además de wifi. 

En este caso me centrare en la estación meteorológica que constara de los siguientes sensores MQ7, DHT11 y el BMP180. Ya en el código 
empezare con una prueba del propio sensor MQ7 (sensor de monóxido de carbono). La segunda prueba será añadiendo el resto de sensores
el DHT11(temperatura y humedad) y el BMP180 (presión y temperatura) además de organizar el código. La tercera prueba será solo 
añadiendo la conectividad wifi. Y por último agregare la conexión con thingspeak para visualizar los datos del MQ7 en la página web. 

Como curiosidad tengo documentado el funcionamiento del primer código y del último código ya que al tratarse de un proyecto 
decidí documentar algunas pruebas.
